# 🧠 MindCare AI

MindCare AI is a Mental Health Support Chatbot developed using HTML, CSS, and JavaScript.

## Features
- 💬 AI Chat
- 😊 Emoji Picker
- 🎤 Voice Input
- 🌙 Dark Mode
- 📄 Export Chat
- 💜 Daily Motivation
- ⚡ Quick Replies

## Technologies Used
- HTML5
- CSS3
- JavaScript
